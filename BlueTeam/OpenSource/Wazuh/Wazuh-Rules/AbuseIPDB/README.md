# AbuseIPDB Integration [![Awesome](https://img.shields.io/badge/SOCFortress-Worlds%20First%20Free%20Cloud%20SOC-orange)](https://www.socfortress.co/trial.html)
> Wazuh, paired with a custom integration script, can be configured to send API requests to AbuseIPDB. Coupled with Wazuh rules, the InfoSec team will immediately be made aware when a known aggressive IP has attempted to SSH into one of their servers.
