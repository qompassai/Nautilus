== Supported Hash Algorithms

The following SHA3-based hash algorithms *MAY* be advertised by this ACVP compliant crypto module:

* SHA3-224 / null / 1.0
* SHA3-256 / null / 1.0
* SHA3-384 / null / 1.0
* SHA3-512 / null / 1.0
* SHA3-224 / null / 2.0
* SHA3-256 / null / 2.0
* SHA3-384 / null / 2.0
* SHA3-512 / null / 2.0
* SHAKE-128 / null / 1.0
* SHAKE-256 / null / 1.0

Other hash algorithms *MAY* be advertised by the ACVP module elsewhere.
