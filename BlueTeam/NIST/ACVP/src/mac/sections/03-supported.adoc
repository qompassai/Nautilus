== Supported HMAC, CMAC, and GMAC Algorithms

The following Message Authentication Code Algorithms *MAY* be advertised by the ACVP compliant cryptographic module:

* HMAC-SHA-1
* HMAC-SHA2-224
* HMAC-SHA2-256
* HMAC-SHA2-384
* HMAC-SHA2-512
* HMAC-SHA2-512/224
* HMAC-SHA2-512/256
* HMAC-SHA3-224
* HMAC-SHA3-256
* HMAC-SHA3-384
* HMAC-SHA3-512
* CMAC-AES
* CMAC-TDES
* ACVP-AES-GMAC
